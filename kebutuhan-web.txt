KEBUTUHAN UNTUK DI DALAM WEBSITE LOKER IT
- Memiliki halaman admin dan user
- Pada halaman admin harus memili dashboard yang diantaranya ada Profile, Lihat loker, Tambah loker, Hapus loker
- Pada halaman user (home) harus menampilkan semua lowongan yang telah ditambahkan oleh admin
- Pada halaman login harus ada username dan password untuk masuk bisa melamar lowongan kerja di web loker it
- Pada halaman register harus ada username, password, nama lengkap, alamat, email, no hp
- Pada halaman user harus memiliki dashboard untuk melihat apa saja yang sudah di apply pada loker tersebut
